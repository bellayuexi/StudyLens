# StudyGraph 用户指南

## 系统简介

StudyGraph 是一个面向中学生的 **AI 驱动知识图谱学习系统**。它帮助你将学习笔记转化为结构化的知识点，通过 AI 问答深入探索每个知识点，并自动生成精美的专题页面，形成个人知识库。

### 核心定位
- **知识管理**：将碎片化笔记转化为结构化知识点，自动分类、打标签、建立关联
- **深度学习**：AI 生成针对性问题，引导你从概念、原因、影响、对比、思考五个维度深入理解
- **知识可视化**：自动生成美观的 HTML 专题页，支持版本管理、批注修改、导出分享

---

## 功能详解

### 1. 知识录入

打开系统后，你可以通过顶部的录入面板输入学习笔记：
- 支持直接粘贴文本（课堂笔记、教材内容等）
- 支持上传文件（PDF、Word、Excel、TXT）
- 支持输入网页 URL，自动抓取内容
- 可选设置"最多提取知识点数"来控制提取粒度（留空则不限制）
- AI 会自动分析并提取知识点，每个知识点包含标题、内容、学科分类和标签
- 自动检测重复知识点，跳过已有内容

**学科分类规则**：历史类会按具体朝代分类（如"历史-北宋"、"历史-唐朝"），数理化等按具体方向分类（如"数学-代数"、"物理-力学"）。

### 2. 知识浏览

系统提供多种浏览方式：
- **分类视图**：按学科分类查看知识点
- **时间线视图**：按时间顺序查看
- **知识图谱**：以图谱形式查看知识点之间的关联关系

### 3. 知识探索（探索更多 Tab）

点击任一知识点进入详情页，切换到"探索更多"标签：

- **智能问题生成**：AI 为你生成 5 个深度学习问题，覆盖概念、原因、影响、对比、思考五个维度
- **批量提问**：勾选感兴趣的问题，一键批量提问，AI 逐个回答
- **自定义提问**：在列表末尾添加你自己想问的问题
- **答案编辑**：对 AI 回答不满意？可以手动编辑或让 AI 重新生成
- **问答删除**：不需要的问答可以直接删除

### 4. 专题页（专题页 Tab）

基于你的问答探索，系统可以生成一个完整的知识专题页：

**生成方式**：
- 通过"智能问答探索"回答一系列问题后生成（推荐，内容更丰富）
- 直接输入学习需求，快速生成

**版本管理**：
- 每次生成或更新都会创建新版本（v1, v2, v3...）
- 可以在版本之间切换查看
- 支持合并两个版本的内容
- 不需要的版本可以删除

**编辑功能**：
- **批注模式**：添加修改建议，AI 根据批注局部更新内容
- **直接编辑**：像编辑网页一样直接修改内容，可删除整个区块
- **导出 HTML**：将专题页导出为独立 HTML 文件

**增量更新**：探索新问题后，专题页会提示"用新回答更新"，AI 会将新内容融入现有页面，而非重新生成。

### 5. 深入分析

对于复杂知识点，可以进行"深入分析"：
- AI 将知识点拆解为 5-8 个子知识点（如"王安石变法"→ 背景、青苗法、免役法等）
- 每个子知识点都是独立的知识卡片，可以单独探索和生成专题页
- 支持导出整个深入分析为单个 HTML 文件（含侧边导航栏）

### 6. 知识重组

你可以用自然语言指示 AI 重新组织知识点：
- 调整分类和标签
- 合并重复知识点
- 批量修改学科分类

## 使用技巧

1. **先探索再生成**：建议先通过"探索更多"回答几轮问题，再生成专题页，这样内容更丰富
2. **控制提取粒度**：对于较长的笔记，设置"最多提取知识点数"（如 5-10 个）可以得到更高层概念，避免过于细碎
3. **善用批注**：生成专题页后，如果某个章节需要补充，使用批注功能比重新生成更精确
4. **利用深入分析**：对于大的知识主题，先做深入分析拆解子主题，逐个击破
5. **定期导出**：重要的专题页可以导出 HTML 备份，支持打印为 PDF（自动处理分页避免截断）

## 未来发展方向

- **多人协作**：支持多个同学共同维护知识库
- **学习路径推荐**：基于已有知识图谱，智能推荐下一步学习内容
- **考试模拟**：基于知识点自动生成测试题
- **跨学科关联**：更智能地发现不同学科之间的知识联系
- **移动端适配**：支持手机和平板访问
- **离线支持**：支持离线浏览已生成的专题页
