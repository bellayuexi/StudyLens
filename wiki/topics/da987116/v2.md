<h1>📜 王安石变法</h1>
<p class="subtitle">北宋中期一场轰轰烈烈的富国强兵改革 · 宋神宗熙宁年间（1069—1085）</p>

<div class="card">
<h2 class="c1">📖 导语</h2>
<p>北宋中期，国家面临严重的<span class="hl">"三冗"危机</span>——冗官、冗兵、冗费，财政入不敷出，对外战争屡战屡败。在这样的背景下，<span class="hl">王安石</span>在宋神宗的全力支持下，发起了一场涉及政治、经济、军事、教育的全面改革，史称<span class="hl">"王安石变法"</span>（又称"熙宁变法"）。</p>
</div>

<div class="card">
<h2 class="c2">🏛️ 时代背景</h2>
<p><strong>北宋立国以来推行"重文轻武"政策</strong>，虽然有效防止了武将割据，却带来了一系列深层问题：</p>
<table>
<tr><th>危机</th><th>表现</th></tr>
<tr><td><span class="hl">冗官</span></td><td>科举取士过多，官僚机构臃肿，办事效率低下</td></tr>
<tr><td><span class="hl">冗兵</span></td><td>军队数量庞大（禁军超80万），但战斗力低下</td></tr>
<tr><td><span class="hl">冗费</span></td><td>官俸、军费开支巨大，加上"岁币"支出，国库空虚</td></tr>
</table>
<p style="margin-top:12px">此外，土地兼并严重，农民负担沉重，社会矛盾尖锐。此前<span class="hl">范仲淹</span>主持的"庆历新政"已经尝试改革但迅速失败，问题愈加严峻。</p>
<div class="quote">天变不足畏，祖宗不足法，人言不足恤。——王安石</div>
</div>

<div class="card">
<h2 class="c3">⚙️ 核心改革措施</h2>
<p>变法内容可分为<span class="tag">富国</span><span class="tag">强兵</span><span class="tag">取士</span>三大方面：</p>

<h3 style="color:#60a5fa;margin:16px 0 8px">💰 富国之法</h3>
<table>
<tr><th>措施</th><th>内容</th><th>目的</th></tr>
<tr><td><span class="hl">青苗法</span></td><td>政府在青黄不接时向农民提供低息贷款</td><td>抑制高利贷，增加财政收入</td></tr>
<tr><td><span class="hl">募役法（免役法）</span></td><td>以交钱代替服役，官僚地主也须缴纳</td><td>减轻农民差役负担，增加政府收入</td></tr>
<tr><td><span class="hl">方田均税法</span></td><td>重新丈量土地，按土地多少和肥瘠收税</td><td>抑制土地兼并，增加税收</td></tr>
<tr><td>农田水利法</td><td>鼓励兴修水利</td><td>促进农业生产</td></tr>
<tr><td>市易法</td><td>政府设市易务，平价收购滞销商品</td><td>稳定物价，打击大商人垄断</td></tr>
<tr><td>均输法</td><td>统筹各地物资运输，就近采购</td><td>节省开支，避免浪费</td></tr>
</table>

<h3 style="color:#34d399;margin:16px 0 8px">🗡️ 强兵之法</h3>
<table>
<tr><th>措施</th><th>内容</th></tr>
<tr><td><span class="hl">保甲法</span></td><td>农户编为保甲，农闲练兵，维护治安，寓兵于民</td></tr>
<tr><td>保马法</td><td>由民户代养官马，减少政府开支</td></tr>
<tr><td>将兵法</td><td>设专职将领训练军队，改变"兵不识将"的弊端</td></tr>
<tr><td>设军器监</td><td>专门管理武器制造，提高武器质量</td></tr>
</table>

<h3 style="color:#f472b6;margin:16px 0 8px">📚 取士之法</h3>
<ul>
<li><strong>改革科举</strong>：废除诗赋取士，改考<span class="hl">经义策论</span>，注重实际能力</li>
<li><strong>整顿太学</strong>：实行"三舍法"，优秀太学生可免试做官</li>
</ul>
</div>

<div class="card">
<h2 class="c4">📊 成效与失败</h2>
<h3 style="color:#34d399;margin-bottom:8px">✅ 积极成效</h3>
<ul>
<li>政府财政收入大幅增加，国库充盈</li>
<li>一定程度上抑制了大官僚、大地主的兼并势力</li>
<li>农田水利得到发展，耕地面积扩大</li>
<li>军事实力有所增强</li>
</ul>

<h3 style="color:#f87171;margin:16px 0 8px">❌ 失败原因</h3>
<ul>
<li><span class="hl">触犯大官僚、大地主利益</span>，遭到以司马光为首的保守派强烈反对</li>
<li>执行过程中用人不当，部分官吏借机盘剥百姓，<strong>好政策变成了害民之举</strong></li>
<li>变法过于急进，缺乏充分的基层配合</li>
<li>宋神宗去世后，<span class="hl">司马光</span>任宰相，新法被全面废除</li>
</ul>

<div class="quote">变法的根本失败原因：在不改变封建土地制度的前提下，无法从根本上解决社会矛盾。改革触动了统治阶级核心利益，却缺乏足够的社会基础支撑。</div>
</div>

<h2 class="c1">✨ 总结</h2>
<p>王安石变法是<span class="hl">北宋中期</span>为挽救统治危机而进行的一次重大改革。它以<span class="hl">"富国强兵"</span>为目标，涉及财政、军事、教育等多个领域，一度取得显著成效。但由于<span class="hl">触犯大官僚大地主利益</span>、执行中的偏差以及支持者宋神宗的去世，变法最终失败。</p>
<p style="margin-top:12px">尽管如此，王安石变法体现了<strong>改革者勇于担当、敢于突破</strong>的精神，其中许多经济思想（如政府调控市场、国家信贷）在今天看来依然具有超前的智慧。</p>
