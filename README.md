# StudyLens

> 把零散的笔记，变成会生长的个人知识库。

StudyLens 是一个 **AI 驱动的深度学习助手**。粘贴一段笔记，它会自动提取知识点、理清脉络，再帮你层层追问、生成可分享的专题页面。

学到的东西不再是一堆散落的文字，而是一座能浏览、能下钻、能不断补充的知识库。

## 上手

```bash
npm install -g studylens
studylens
```

打开 `http://localhost:3000`，首次启动会自动引导你配置大模型。详细步骤见 **[快速开始](docs/getting-started.md)**。

## 怎么用

典型的学习流程是「录入 → 探索 → 生成 → 深化」：

1. **录入** — 把课堂笔记粘进去让 AI 拆解成知识点，或手动逐条添加。可设"最多提取数"控制粒度。
2. **探索** — 点开任一知识点的「探索更多」，AI 从概念、原因、影响、对比、思考五个维度生成问题，逐个追问、积累理解。
3. **生成专题页** — 基于问答探索，一键生成图文并茂的学习专题；支持版本历史、批注修改、导出 HTML（可打印为 PDF）。
4. **深入分析** — 对大主题做「深入分析」，AI 拆成多个子知识点（如"王安石变法"→ 背景、青苗法、免役法……），逐个击破。

> 小技巧：先在「探索更多」回答几轮问题，再生成专题页，内容会更丰富。完整说明见 **[用户指南](docs/user-guide.md)**。

## 它能做什么

- **两种录入方式** — 粘贴笔记让 AI 拆解，或手动逐条添加知识点
- **自动提取知识点** — AI 识别要点、归类、打标签，并建立彼此的关联
- **层层深入** — 对任一概念继续追问，AI 拆解出子主题，逐个击破
- **专题页面** — 一键生成图文并茂的学习专题，带版本历史，可批注、可导出
- **智能问答** — 围绕你的知识库提问，得到结合上下文的回答
- **多视图浏览** — 按学科分类，或沿时间线回顾
- **完全可控的模型** — 支持智谱、通义千问、DeepSeek、OpenAI 等任意 OpenAI 兼容服务，也支持 Ollama 本地免费运行
